# Deployment Guide

## Hosting on GitHub Pages

1. Push your folder to a GitHub repository.
2. Go to repository settings > Pages > Select branch (`main` or `master`).
3. Site will go live at: `https://yourusername.github.io/reponame/`

## Hosting on Netlify

1. Sign up at [Netlify](https://www.netlify.com/).
2. Click "Add New Site" and connect your GitHub repo.
3. Netlify will auto-deploy your site.

## For Any Updates

- Edit your files locally, commit & push changes to GitHub.
- Site updates automatically.

## Google Analytics

- Paste your Analytics tracking script in `js/analytics.js`.

## Content Management

- Edit HTML files for changing content.
- Blog posts/articles can be added in `blog.html`.
